# Architecture Decision Records (ADR)

This directory holds Architecture Decision Records for the Open Communication Platform.

## When an ADR is required

Per [ENGINEERING_PRINCIPLES.md](../ENGINEERING_PRINCIPLES.md), any implementation that violates a normative engineering principle **must** be documented here with an approved ADR before proceeding.

## Format

Each ADR is a markdown file named `NNNN-short-title.md`:

```markdown
# ADR-NNNN: Title

**Status:** Proposed | Accepted | Deprecated | Superseded
**Date:** YYYY-MM-DD

## Context

## Decision

## Consequences

## Principles affected
```
